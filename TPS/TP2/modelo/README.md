## Modelo en GLPK

Ver anexos del informe para mas detalles.

### Ejecución:

#### Basico

```
glpsol -m tp2.mod -d datos/tp2_basico.dat -o solucion/tp2_basico.sol
```

#### Avanzado

```
glpsol -m tp2.mod -d datos/tp2_avanzado.dat -o solucion/tp2_avanzado.sol
```

### Datos:

Ver `/datos` para su obtención, transformación y parseo

### Soluciones:

Ver `/solucion` para su obtención, transformación y parseo
